Theorem 7.22.

Let \(A\) be a set. Suppose that \(A\) is finite.
  1. If \(X \subseteq A\text{,}\) then \(X\) is finite.
  2. If \(X \subseteq A\text{,}\) then \(|A| = |X| + |A - X|\text{.}\)
  3. If \(X $ A\text{,}\) then \(|X| < |A|\text{.}\)
  4. If \(X $ A\text{,}\) then \(X \not\# A\text{.}\)
