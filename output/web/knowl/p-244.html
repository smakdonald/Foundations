⇐. Suppose that m and n are both odd. Hence there is an integer j such that m = 2 j + 1, and there is an integer k such that n = 2k + 1. Therefore mn = (2 j + 1)(2k + 1) = 4 jk + 2 j + 2k + 1 = 2(2 jk + j + k) + 1. Because k and j are integers, so is 2 jk + j + k. Therefore mn is odd.
