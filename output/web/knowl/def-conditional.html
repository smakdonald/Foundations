Definition 1.11.

Let \(P\) and \(Q\) be statements. The conditional from \(P\) to \(Q\text{,}\) which is denoted \(P \to Q\text{,}\) is the statement that, intuitively, is true if it is never the case that \(P\) is true and \(Q\) is false.
