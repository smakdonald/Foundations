Axiom B.9. Axiom of Replacement.

Let \(F(s,t)\) be a functional property of sets with two free variables \(s\) and \(t\) that can be formulated in the context of the ZF axioms. Let \(x\) be a set. Then there is a set \(z\) such that \(y \in z\) if and only if there is some \(w \in x\) such that \(F(w, y)\) is true.
