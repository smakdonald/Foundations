Item.

Let \(A = \{a, b, c\}\text{.}\) Then the subsets of \(A\) are \(\emptyset, \{a\}, \{b\}, \{c\}, \{a, b\}, \{a, c\}, \{b, c\} and \{a, b, c\}\text{.}\) The last of these subsets is not proper, but we need all subsets, not only the proper ones. Therefore
\begin{equation*} \cP(A) = \{ \emptyset, \{a\}, \{b\}, \{c\}, \{a, b\}, \{a, c\}, \{b, c\}, \{a, b, c\}\}. \end{equation*}
It can be seen intuitively that if \(A\) is a finite set with n elements, then \(\cP(A)\) is a finite set with \(2^n\) elements; by Part (1) of this exercise we see that this formula holds even when \(n = 0\text{.}\) This formula is proved in Theorem 7.7.10 (1).
