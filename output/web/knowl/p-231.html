Formally, we use proof by cases when the premise P can be written in the form A ∨ B. We then use Exercise 1.3.2 (6) to see that (A ∨ B) → Q is equivalent to (A → Q) ∧ (B → Q). Hence, in order to prove that a statement of the form (A ∨ B) → Q is true, it is sufficient to prove that each of the statements A → Q and B → Q is true.
