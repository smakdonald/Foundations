Let \(A\text{,}\) \(B\) and \(C\) be sets.
  1. \(A \cap B \subseteq A\) and \(A \cap B \subseteq B\text{.}\) If \(X\) is a set such that \(X \subseteq A\) and \(X \subseteq B\text{,}\) then \(X \subseteq A \cap B\text{.}\)
  2. \(A \subseteq A \cup B\) and \(B \subseteq A \cup B\text{.}\) If \(Y\) is a set such that \(A \subseteq Y\) and \(B \subseteq Y\text{,}\) then \(A \cup B \subseteq Y\) .
  3. Commutative Laws.
    \(A \cup B = B \cup A\) and \(A \cap B = B \cap A\text{.}\)
  4. Associative Laws.
    \((A\cup B)\cup C = A\cup (B\cup C)\) and \((A\cap B)\cap C = A\cap (B\cap C)\text{.}\)
  5. Distributive Laws.
    \(A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\) and \(A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\text{.}\)
  6. Identity Laws.
    \(A \cup\emptyset = A\) and \(A \cap\emptyset = \emptyset\text{.}\)
  7. Idempotent Laws.
    \(A \cup A = A\) and \(A \cap A = A\text{.}\)
  8. Absorbtion Laws.
    \(A \cup (A \cap B) = A\) and \(A \cap (A \cup B) = A\text{.}\)
  9. If \(A \subseteq B\text{,}\) then \(A \cup C \subseteq B \cup C\) and \(A \cap C \subseteq B \cap C\text{.}\)
