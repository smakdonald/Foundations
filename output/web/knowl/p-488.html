Let \(A\) and \(B\) be non-empty sets, and let \(f : A \to B\) be a function.
  1. The function \(f\) is injective if and only if \(f \circ g = f \circ h\) implies \(g = h\) for all functions g\(, h : Y \to A\) for all sets \(Y\) .
  2. The function \(f\) is surjective if and only if \(g \circ f = h \circ f\) implies \(g = h\) for all functions \(g, h : B \to X\) for all sets \(X\text{.}\)
